# CNAB 240

Esta é uma implementação do Layout Padrão Febraban 240 posições, baseada no BinData. 

## Instalação

Adicione ao Gemfile:

    gem 'cnab240'

execute:

    $ bundle

Ou adicione no seu chiqueiro:

    $ gem install cnab240
